哎呀，你想问的是\`pygame\.display\.flip\(\)\`或者\`pygame\.display\.update\(\)\`吧，这两个函数是用来更新显示屏幕的。在Pygame中，画图和填充颜色的操作不会立即显示到屏幕上，而是要通过这些函数来刷新画面。

我们来看一下具体的用法：

\`\`\`python
import pygame

\# 初始化pygame
pygame\.init\(\)

\# 设置窗口大小
screen = pygame\.display\.set\_mode\(\(800, 600\)\)

\# 定义颜色（RGB格式）
WHITE = \(255, 255, 255\)
BLACK = \(0, 0, 0\)

\# 渲染循环标志
running = True

while running:
for event in pygame\.event\.get\(\):
if event\.type == pygame\.QUIT:
running = False

\# 填充屏幕背景色为白色
screen\.fill\(WHITE\)

\# 更新显示（刷新画面）
pygame\.display\.flip\(\)

\# 退出pygame
pygame\.quit\(\)
\`\`\`

在这段代码中，\`pygame\.display\.flip\(\)\`会将绘制的内容更新到屏幕上。你也可以使用\`pygame\.display\.update\(\[rect\]\)\`来指定要更新的区域或全屏更新。

比如：
\`\`\`python
\# 填充屏幕背景色为黑色

\# 或者只更新部分区域，假设有一个矩形rect = pygame\.Rect\(10, 20, 50, 60\)
