350234最戳人的，从不是三国背景的壳子，而是把“爱错人”的绝望，揉进了两个本该只谈权力的男人骨血里。你看吕布多傻啊，他本是能在沙场上挑翻千军的将才，却把董卓当成了余生的归处——破落山庙里的一次纠缠，对董卓或许只是权宜之计，对吕布却是心湖炸开的惊雷。他亲手杀了丁原，背着重义的骂名投奔过去，夜里躺在同一张榻上时，他盯着董卓的睡颜，大概以为抓住了世上唯一的光；可董卓呢，他摸吕布的头发、听吕布说情话时，眼里装的从来不是爱意，是“这个人能为我打天下”的算计。<br>后来貂蝉来了，那根本不是什么“第三者”，而是照妖镜。董卓立刻被新鲜感和美色勾走，忘了吕布夜里会悄悄为他盖好被子，忘了他曾说“有你在便够了”。最疼的不是被冷落，是吕布红着眼眶问“你怎么不看我了”时，董卓眼里的不耐烦；是误会发生后，董卓拔剑砍向他肩膀时，那毫不犹豫的狠劲。那一刀砍破的哪里是皮肉，是吕布最后一点可怜的期待——他终于看清，自己在董卓心里，连权力路上的一块垫脚石都不如。<br>更让人心里发紧的，是两人从头到尾的“错位”。吕布在床上是强势的，可感情里却像个没安全感的小孩，他要的是“你只对我好”的承诺；董卓在床上依赖吕布的温度，可心里的帝王梦从来没停过，他给的“温柔”全是带着钩子的诱饵。直到刑场上，吕布提着染血的剑问“你有没有话要对我说”，董卓的沉默才是最狠的回答——他到死都没懂，吕布要的从来不是什么高官厚禄，只是一句真心的“我曾对你动过心”。<br>最后吕布看着董卓的尸体倒下时，他心里该是空荡荡的吧？他赢了权力的棋局，却输了自己唯一想护的人。漫画里那些细腻的画功多残忍啊，两人初遇时的暖黄灯光、吕布被砍伤后攥紧的拳头、刑场上飘着的冷雪，每一笔都在放大这种“求而不得”的痛。其实我们都知道这是虚构的，可还是会为吕布难过——原来再厉害的人，一旦动了真心，也会变成任人宰割的傻瓜，而错付的真心，最后只会变成插在自己心上的刀，拔出来是血，不拔是疼。
